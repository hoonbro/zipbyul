이번 버전은 "바로 개발 들어갈 수 있게" 모호한 범위를 결정으로 바꾼 것이 핵심이다.

| 항목 | v1.1 | v2.0 확정 |
| ---- | ---- | ---- |
| 집값전망 심리 데이터원 | 한국은행 CSI(전국/지역 미확정) | **ECOS 서울 주택가격전망 CSI**로 확정. 서울 단위 시계열이 별도 제공됨을 확인 |
| 실거래 알림 | 알림 카테고리에 포함(자치구 단위) | **기본 푸시 OFF**. 자치구 단위는 noise가 크므로 "관심지역 화면 요약"이 기본. 즉시 푸시는 사용자가 명시적으로 켠 경우에만(옵트인) |
| 정책·금리 일정 데이터 | "주요 정책 일정" 폭넓게 | **MVP는 한국은행 기준금리 결정일로 한정**. 백오피스가 2차라 수동 정책 일정은 넣을 경로가 없음 → seed 데이터로만 채움 |
| 관심지역 단위 | 자치구만 | 자치구 기본 + 실거래에 한해 **법정동 1~2개 좁히기(선택)** 허용 |
| 비회원 사용 | 도입 | 유지 (기기/브라우저 익명 식별자) |
| 시장온도·전세온도, 뉴스, 페르소나/8문답, 백오피스 | 2차 | 2차 유지 |

### v2.1 추가 — 목업 정합화 결정

목업 v2.1과 어긋나던 4개 지점 중, **D만 기획안 기준으로, A·B·C는 목업 기준으로** 확정했다.

| # | 지점 | 확정 |
| - | ---- | ---- |
| A | 온보딩 관심유형 | **목업 9종 채택**(민간청약·무순위 청약·행복주택·청년안심주택·장기전세·매입임대·정책·금리·실거래 등록·집값 전망 심리). 단 **청년안심주택·장기전세는 데이터원이 2차(SH)**라 선택은 되지만 MVP에는 콘텐츠가 없고 "개발 예정"으로 표시 |
| B | 통합 캘린더 분류 | **목업 분류 채택**(청약: 민간분양·무순위 / 공공: 행복주택·청년안심주택·장기전세·매입임대 / 정책: 기준금리·청약제도). 단 **MVP에 실제 채워지는 소스는 청약홈·LH·기준금리**뿐. SH·비정기정책 분류는 데이터 2차(빈 상태/개발 예정) |
| C | 알림 수준 | **목업 5종 채택**(모든 알림 / 중요 알림만=**별 3개↑** / 마감 임박만=D-3 / 관심지역만 / 하루 요약만). "전체 끄기"는 알림 수준이 아니라 푸시 권한·기기 토글(`push_enabled=false`)로 처리 |
| D | 집값전망 심리 | **기획안 기준 = ECOS 서울 주택가격전망 CSI** 유지 |

> A·B의 핵심: "관심유형/캘린더 분류는 목업 UI를 따르되, **데이터원이 2차인 항목(청년안심주택·장기전세·비기준금리 정책)은 선택 가능하나 MVP에선 빈 상태/개발 예정으로 표시**"한다. 이는 목업이 지수 카드에서 이미 쓰는 "제공 중 / 개발 예정" 방식과 동일하다.

## 1. 서비스 개요

### 1-1. 서비스명 · 부제

**집별** — 서울 부동산 레이더

### 1-2. 한 줄 소개

서울의 청약·공공임대 일정, 기준금리 결정 일정, 관심지역 실거래 신규 등록, 한국은행 서울 주택가격전망 CSI 기반 집값전망 심리를 한곳에 모으고, 사용자에게 중요한 정보만 알림으로 알려주는 비회원 기반 서울 부동산 정보 레이더 앱.

### 1-3. 콘셉트

경제 앱 BYUL이 "중요한 변화만 모아 알려주듯", 집별은 서울 부동산에서 놓치면 안 되는 일정과 변화를 수집·분류·정리해 전달한다. 매물 검색 앱이 아니라, **놓치지 않게 해주는 알림형 리서치 앱**이다.

> 집별이 답하는 질문
> - 내 관심지역(서울 자치구)에 새 청약·공공임대 공고가 생겼나?
> - 접수 시작·마감 임박·당첨자 발표 같은 중요한 일정을 놓치고 있지 않나?
> - 기준금리 결정이나 서울 집값 전망 심리가 바뀌었나?
> - 관심지역에 실거래 신규 등록이 있었나?

------

## 2. 해결하려는 문제

1. **출처가 흩어져 있다.** 청약홈, LH, 한국은행, 국토부 실거래가를 따로 확인해야 한다.
2. **일정성 정보는 놓치면 손해가 크다.** 접수일·서류제출일·당첨자 발표일은 놓치면 기회가 사라진다.
3. **금리·정책은 어렵고 흩어져 있다.** 기준금리 일정조차 일반 사용자는 추적하기 어렵다.
4. **기존 앱은 매물·시세 중심이다.** 청약·공공임대·금리·실거래 등록을 관심지역 기준으로 알림까지 연결하는 경험은 약하다.
5. **시장 심리를 한눈에 보기 어렵다.** 집별은 ECOS 서울 주택가격전망 CSI를 "집값 전망 심리"로 쉽게 보여준다.

------

## 3. 목표 사용자

### 3-1. 1차 핵심 타깃

**서울에서 내 집 마련 또는 이사를 준비하는 사용자** — 서울 거주자·진입 희망자, 청약·공공임대 대기자, 실거주 매수 검토자, 전월세 이사 준비자.

### 3-2. MVP 개인화 방식

페르소나·8문답을 쓰지 않는다. **관심지역(서울 자치구) + 관심유형 + 알림수준** 3가지만으로 개인화한다.

| 사용자 유형 | 핵심 니즈 | MVP 제공 |
| ---- | ---- | ---- |
| 청약·분양 관심자 | 민간분양·무순위·오피스텔 접수 일정 | 청약홈 기반 캘린더, D-Day 알림, 원문 링크 |
| 공공임대 관심자 | LH 행복주택·국민임대·매입임대 공고 | LH 기반 공공임대 캘린더, 마감 알림 |
| 실거주 매수 검토자 | 금리 일정, 집값 전망 심리, 관심지역 실거래 | 기준금리 일정, 서울 집값 전망 심리, 실거래 요약 |
| 전월세 이사 준비자 | 공공임대 기회, 관심지역 전월세 등록 | 공공임대 일정, 실거래 요약, 마감 임박 알림 |
| 시장 모니터링 사용자 | 주요 일정을 짧게 파악 | 오늘의 레이더, 중요도 별점, 하루 요약 |

### 3-3. MVP에서 제외하는 타깃

단기 투자자·갭투자자·시세 트레이딩 목적. 집별은 투자 판단을 대신해주는 앱이 아니다.

## 4. 핵심 가치 제안

1. **흩어진 공식 정보를 한곳에.** 청약홈·LH·국토부 실거래가·한국은행 ECOS 등 공식 OpenAPI 기반 정보를 통합한다.
2. **관심지역·관심유형 기준으로 보여준다.** 선택한 서울 자치구와 유형에 맞춰 우선 노출한다.
3. **공고 핵심 정보를 먼저 보여준다.** 긴 공고문 대신 일정·지역·공급유형·접수기간·당첨자 발표일·원문 링크를 구조화해 제공한다.
4. **중요한 변화만 알림.** 신규 공고, 접수 시작, 마감 임박, 당첨자 발표, 기준금리 결정을 중요도·알림수준에 따라 푸시한다.
5. **집값전망 심리를 단순하게.** ECOS 서울 주택가격전망 CSI를 "집값 전망 심리"로 표시한다.

### 차별화 한 줄

청약 알리미가 "일정"만, 시세 앱이 "가격"만 본다면, 집별은 **청약·공공임대 일정 + 기준금리 일정 + 관심지역 실거래 신규 등록 + 서울 집값 전망 심리를 한 화면에 묶고, 중요도·관심지역 기준으로 걸러 알림으로 연결**한다. 핵심 hero 경험은 **"매일 아침 하루 요약 1건"**이다.

## 5. 최종 MVP 방향

초기 검증 목표:

> - 서울 부동산 관심자가 매일/매주 들어올 이유가 있는가?
> - 직접 검색하지 않아도 놓치면 안 되는 일정·변화를 받을 수 있는가?
> - 공식 API 기반 데이터만으로 알림형 레이더 경험이 성립하는가?

MVP는 아래 5개 축에 집중한다.

- **축 A. 관심지역 개인화** — 서울 자치구 단위. 해당 지역 일정·공고·실거래를 우선 제공.
- **축 B. 통합 캘린더 + D-Day 알림** — 청약홈·LH 청약/공공임대 일정 + 한국은행 기준금리 결정일.
- **축 C. 공고 핵심 정보 정리** — 구조화 필드 우선. AI 요약은 선택 기능.
- **축 D. 관심지역 실거래 신규 등록** — 지도·관심단지·가격분석 없이 "신규 등록 요약"만. 신고지연 고지 필수. **기본 푸시 OFF**.
- **축 E. 집값 전망 심리** — ECOS 서울 주택가격전망 CSI.

### 2차로 분리한 항목

| 항목 | 2차로 미루는 이유 |
| ---- | ---- |
| 집별 시장온도 / 전세 온도 | 국토연구원 지수 API·상업 라이선스·산식 검증 필요 |
| 뉴스 RSS/AI 요약 | 저작권·요약 품질·중복 제거 부담 |
| SH·서울주거포털·청년안심주택 웹 모니터링 | 공식 API 불명확, 수동 검수 부담 |
| 백오피스 | 운영자 기능. MVP 검증 후 도입 |
| 페르소나/8문답 | 초기 개인화에 과도 |
| 관심단지 추적 | 단지명 정규화·중복·법정동 매칭 난이도 |
| 폭넓은 정책 일정 | 비정기·수동 데이터라 백오피스(2차) 필요. MVP는 기준금리 결정일만 |

## 6. MVP 핵심 기능

### 6-1. 온보딩 (비회원 기본)

회원가입 없이 시작한다. 기기/브라우저 기준 익명 식별자(`anonymousId`)에 설정을 저장한다.

1. **관심지역** — 서울 자치구 1~5개 선택. (실거래에 한해 자치구 내 법정동 1~2개 좁히기 선택 가능)
2. **관심유형 (목업 9종)** — 민간청약 / 무순위 청약 / 행복주택 / 청년안심주택 / 장기전세 / 매입임대 / 정책·금리 / 실거래 등록 / 집값 전망 심리
   - **청년안심주택·장기전세**는 데이터원이 SH(2차)라 선택은 되지만 MVP에는 콘텐츠가 없다 → 칩에 "개발 예정" 표시, 알림·피드에서 빈 상태.
   - 코드값·매칭 규칙(어떤 supply_type/event에 매칭되는지)은 백엔드 기획안 부록 A를 SSOT로 한다.
3. **알림 수준 (목업 5종)** — 모든 알림 / 중요 알림만(별 3개↑) / 마감 임박만(D-3) / 관심지역만 / 하루 요약만. (전체 끄기는 알림 설정 화면의 푸시 마스터 토글로 처리)

> 페르소나·상황 선택·8문답은 사용하지 않는다.

### 6-2. 홈 피드 — 오늘의 집별 레이더

앱을 켰을 때 오늘 확인할 서울 부동산 일정·변화를 30초 안에 파악하게 한다.

구성: ① 오늘 스캔 요약 ② 집값 전망 심리 카드 ③ 기준금리·정책 일정 카드 ④ 마감 임박 일정 ⑤ 관심지역 신규 공고 ⑥ 관심지역 실거래 신규 등록(요약) ⑦ 관심지역 요약 ⑧ 중요도 별점 ⑨ 데이터 출처·갱신시각·신고지연 고지.

```
오늘의 집별 레이더

[집값 전망 심리]  서울 102 / 상승 전망 우세
출처: 한국은행 서울 주택가격전망 CSI · 기준월 2026.06

[중요 일정]
- 강동구 무순위 청약 접수 오늘 시작
- LH 행복주택 접수 마감 D-2
- 한국은행 기준금리 결정 D-1

[내 관심지역]
- 마포구: 공공임대 1건 / 실거래 신규 등록 4건 / 마감임박 1건
- 성동구: 청약 0건 / 실거래 신규 등록 2건
```

### 6-3. 집값 전망 심리

- **데이터원**: 한국은행 ECOS **서울** 주택가격전망 CSI (소비자동향조사)
- **표현**: "집값 전망 심리"
- **해석 기준**: 100 미만 = 하락 전망 우세 / 100 부근 = 중립 / 100 초과 = 상승 전망 우세
- **활용**: 홈 상단 카드, 월별 추이, 급변 시 하루 요약/중요 알림 후보, 출처·기준월·갱신일 표시
- **주의 문구**: 가격 예측·투자 추천이 아니라 시장 참여자의 기대 심리를 이해하기 위한 참고 지표. 실제 가격 변동과 다를 수 있음.
- **출처 표시 의무**: ECOS 이용약관상 상업 이용 시 출처표시가 전제이므로 카드·상세에 "한국은행 ECOS" 출처를 항상 노출한다.

### 6-4. 통합 캘린더

청약·공공임대·기준금리 일정을 한 캘린더에서 본다.

**캘린더 분류·필터 (목업 기준)**: 청약(민간분양·무순위) / 공공(행복주택·청년안심주택·장기전세·매입임대) / 정책(기준금리·청약제도). 일정 단계(stage)는 모집공고·접수 시작·접수 마감·서류제출·당첨자 발표·계약·정책 발표.

**MVP에 실제 채워지는 소스**: 청약홈(민간분양·무순위·오피스텔·공공분양), LH(행복주택·국민임대·매입임대·전세임대), **한국은행 기준금리 결정일**. → 분류 UI는 목업 전체를 노출하되, **청년안심주택·장기전세(SH=2차)와 청약제도 등 비기준금리 정책(2차)은 MVP에서 데이터가 없어 빈 분류/개발 예정으로 표시**한다.

> MVP 정책 일정은 기준금리 결정일로 한정한다. 청약제도 변경 등 비정기 정책은 백오피스(2차) 도입 후 채운다. 캘린더 항목은 공고 날짜 필드(모집공고일·접수일·서류제출일·당첨발표일·계약일)에서 단계별로 생성한다.

**일정 상세**: 공고명, 지역, 공급유형, 모집공고일, 접수 시작/마감일, 당첨자 발표일, 계약일, 신청 대상, 핵심 자격 필드, 원문 링크, 데이터 출처, 마지막 수집 시각.

### 6-5. 공고 핵심 정보 정리

긴 공고문을 빠르게 이해하게 한다. 누가/어디/몇 세대/신청기간/서류·발표일/소득·자산·무주택 조건/반드시 확인할 부분/원문 링크.

**AI 요약 정책**: 선택 기능(필수 아님). 일정·금액·세대수·날짜·자격 기준 등 수치는 LLM이 생성하지 않고 원문 구조화 필드를 사용. "참고용이며 최종 신청 전 원문 확인 필요" 문구 고정. AI 실패 시 구조화 필드만으로 동작.

### 6-6. 관심지역 실거래 신규 등록

선택한 관심지역에 새 실거래가 등록되면 보여준다.

**MVP 범위**: 실거래 지도, 관심단지 추적, 가격 예측, 신고가·급락 분석은 제공하지 않는다. 제공: ① 관심지역 신규 실거래 등록 목록 ② 매매/전월세 구분 ③ 면적·거래금액·계약월·층수 기본 정보 ④ 최근 데이터 불완전 고지 ⑤ 출처·마지막 수집 시각.

**알림 위상 (v2.0 변경)**: 자치구 단위 실거래는 건수가 많아 noise가 크다. 따라서 **기본은 푸시 OFF, "관심지역 화면 요약"으로만 노출**한다. 즉시 푸시는 (a) 사용자가 실거래 알림을 명시적으로 켜고 (b) 자치구 내 법정동 1~2개로 좁힌 경우에만 후보가 된다. 그 외에는 하루 요약에만 포함한다.

**표현 주의**: "실시간 거래 발생" 금지. "신규 실거래 등록 / 최근 등록된 실거래 / 신고 지연으로 최근 데이터 일부 누락 가능" 표현 사용.

### 6-7. 알림 설정

**알림 카테고리**: 신규 청약 공고, 공공임대 모집공고, 접수 시작, 마감 임박, 당첨자 발표, 기준금리 일정, 관심지역 실거래 등록(기본 OFF), 집값 전망 심리 변화, 오늘의 요약 리포트.

**알림 수준**(서버 enum, 목업 5종): 모든 알림 `ALL` / 중요 알림만 `IMPORTANT_ONLY`(**별 3개↑ = final_score ≥ 4**) / 마감 임박만 `DEADLINE_ONLY`(D-3·D-1·당일) / 관심지역만 `REGION_ONLY` / 하루 요약만 `DAILY_DIGEST_ONLY`. "전체 끄기"는 별도 enum이 아니라 푸시 마스터 토글(`push_enabled=false`)로 처리한다.

**방해금지 모드**: 야간/주말 알림 끄기, 긴급 알림만, 하루 요약 시간 설정. 시간 기준은 모두 `Asia/Seoul`.

## 7. 중요도 별점 기준 (요약)

> 상세 산정 룰과 "이벤트 base 점수 / 사용자별 개인화 점수" 분리는 백엔드 기획안 6장을 SSOT로 한다. 본 문서는 사용자 노출 기준만 정의한다.

중요도는 0~10점 규칙 기반으로 계산하고, 화면에는 1~5 별점으로 변환한다. MVP에서 AI는 중요도를 산정하지 않는다.

| 중요도 점수 | 별점 | 알림 기본 판단 |
| ---- | ---- | ---- |
| 0~1 | ★☆☆☆☆ | 알림 제외, 목록 하단 |
| 2~3 | ★★☆☆☆ | 하루 요약 중심 |
| 4~5 | ★★★☆☆ | 보통 이상 알림 대상 |
| 6~7 | ★★★★☆ | 중요 알림 대상 |
| 8~10 | ★★★★★ | 최우선 알림 대상 |

## 13. 성공 지표 (KPI)

- 온보딩 시작자의 70%↑가 관심지역 설정
- 온보딩 완료자의 60%↑가 관심유형 2개↑ 선택
- 푸시 허용률 30%↑, 클릭률 10%↑
- 주간 재방문율 35%↑
- 관심지역 알림 수신자의 재방문율이 일반보다 높음
- 홈 방문자의 40%↑가 캘린더 또는 공고 상세로 이동
- (데이터) 청약홈/LH/실거래/ECOS 수집 성공률, 공고·실거래 반영 속도, 중복 제거 정확도, 알림 발송 정확도, 출처·갱신시각 표시 누락률

## 14. 최종 MVP 정의

집별의 1차 MVP는 서울에서 내 집 마련/이사를 준비하는 사용자가 **청약·공공임대 일정, 기준금리 일정, 관심지역 실거래 신규 등록, 서울 집값 전망 심리**를 한곳에서 확인하고, 중요한 일정·변화만 알림으로 받는 **비회원 기반 부동산 정보 레이더 앱**이다.

핵심 6가지:

1. 비회원 기반 관심지역·관심유형 설정
2. 청약홈·LH 기반 청약·공공임대 통합 캘린더 + 기준금리 결정일
3. ECOS 서울 주택가격전망 CSI 기반 집값 전망 심리
4. 국토부 실거래가 기반 관심지역 신규 등록 요약 (기본 푸시 OFF)
5. 중요도/별점 기반 D-Day·마감임박 푸시 알림 + 하루 요약
6. 데이터 출처·갱신시각·신고지연 고지를 포함한 신뢰성 표시

집별은 매물 앱이 아니라, 서울 부동산 정보를 놓치지 않게 해주는 알림형 리서치 앱으로 포지셔닝한다.
